## This package contains launch file and python script for semi-autonomous navigation
   for the taylor-Dunn vehicle

To apply algorithm in simulation, run the following code with taylor_navi/src folder

python follow_wall_sim.py
python user_input.py

To apply algorithm on real cart, launch navi.launch (with all sensors connected)
and runfollow_wall_real.py instead

You should be able to enter w,a,s,d commands and control the vehicle
